# Hardware Adapter Contracts

This directory describes AgenticOS hardware and middleware adapter contracts.

Concrete ROS2 bridge ownership, generated adapter config, and lifecycle
metadata belong under `/opt/agentic/bridges/ros2`. ROS2 bridge source packages
may be staged under `/home/ubuntu/agentic_ws/ros2_bridge_src`, but that path is
a colcon source/build workspace, not an Agent App workspace.
